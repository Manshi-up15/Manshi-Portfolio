ExamGuard

An anti-cheating seat allocation system powered by the
N-Queens backtracking algorithm

🎓
The Problem: In any exam hall, students sitting in the same row, column, or diagonal can see each other's answer sheets — exactly how a chess queen threatens other pieces. The N-Queens algorithm places N students across an N×N hall so that no two share a line of sight — making cheating geometrically impossible.

How to Play
Manual Mode: Click any cell to place a student. The hall highlights threat zones in real time. Place one student per row. If a conflict is detected, the cell turns red — backtrack and try again!

Auto-Solve: Watch the backtracking algorithm find all valid solutions with step-by-step animation. Use Next/Prev to browse solutions.
