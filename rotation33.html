Understanding Rotation

Rotation Transformation Explained

Unlike scaling (which changes size) or translation (which moves position), rotation is a pure orientation change. Every point in the 3D mesh is repositioned around a fixed axis using trigonometry.

We use Euler angles combined with a quaternion slerp to avoid gimbal lock — the smooth, continuous rotation you feel when dragging is mathematically precise angular transformation.

Rx(θ) = [1, 0, 0 ]
        [0, cos θ, -sin θ ]
        [0, sin θ, cos θ ]

Ry(φ) = [ cos φ, 0, sin φ ]
        [ 0, 1, 0 ]
        [-sin φ, 0, cos φ ]
X Axis — Pitch (tilt up/down)
Y Axis — Yaw (spin left/right)
Z Axis — Roll (lean side)
