N-Queens · Real World
Real-Life Problem: In an exam hall, students sitting in the same row, column, or diagonal can easily see each other's answer sheets — just like how a chess queen attacks. The N-Queens algorithm places N students in an N×N hall such that no two students share a row, column, or diagonal — making cheating geometrically impossible.

How It Works
  • 01 Place first student in row 1. Try each column.
  • 02 For each placement, check: same column? Same diagonal?
  • 03 If safe → move to next row (recurse deeper).
  • 04 If conflict → backtrack, try next column in current row.
  • 05 Repeat until all N students are placed with 0 conflicts.
  • 06 Each valid arrangement = one safe exam seating plan.
